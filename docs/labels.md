# Labels

Use these labels during closed testing:

- `needs-review` - maintainer triage required.
- `approved-scrape` - maintainer approved a private manual scrape/export attempt.
- `source-needed` - public source URL or source detail is missing.
- `verification` - issue contains field verification evidence.
- `blocked-ip` - source material is private, copied, restricted, or otherwise unusable.
- `manufacturer-correction` - manufacturer or representative correction request.
