# Manufacturer Posture

AI4AV publishes control specs derived from public manufacturer documentation and field verification.

## Source Policy

- Prefer first-party manufacturer documentation.
- Use third-party mirrors only when the manufacturer document is clearly identified and no better public source is available.
- Do not publish private manuals, NDA material, partner-only downloads, credentials, or copied commercial driver data.
- Keep provenance visible in each exported spec.

## Corrections And Removal

Manufacturers can file a correction issue with the affected spec ID, the correct public source, and the requested change. Maintainers will review and update, deprecate, or remove the affected catalog entry as appropriate.

If a published spec accidentally includes non-public material, file a private security report rather than a public issue.
