# Sharp PN-R Series Control Spec

## Summary
Sharp PN-R Series professional LCD monitors support RS-232C and LAN control protocols for power management, input selection, display adjustment, audio control, and multi-monitor configuration. Daisy chain control supports up to 25 monitors via RS-232C. LAN control uses Telnet with username/password authentication.

## Notes
**Command format:** 4-char command + 4-char parameter (space-padded), terminated by CR/LF. Negative values use 3-digit signed format (e.g., AUTR-005).

**Daisy chain:** Up to 25 monitors via RS-232C; ID-based targeting or repeater control (4th char = "+"). Response timeout = monitor position × 10s minimum.

**WAIT responses:** Commands RSET, INPS, ASNC, WIDE, EMAG, EPOS, PXSL, POWR, AGIN, MWIN, MWIP, MWPP, ESTG, EMHV, EPHV, ESHV, ENLG, DPST return WAIT and require no further commands until response.

**LAN control:** Web browser (HTTP) or Telnet. Telnet uses same RS-232C commands over TCP socket after login handshake. Default credentials: admin/admin. DATA PORT range 1025-65535.

**Locked response:** Returns "LOCKED" when RS-232C control is locked via operation lock function.
