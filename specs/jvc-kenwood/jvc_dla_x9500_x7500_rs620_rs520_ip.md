# JVC KENWOOD DLA-X9500/X7500/RS620/RS520 Control Spec

## Summary
JVC KENWOOD D-ILA projector control spec covering RS-232C and LAN control on the DLA-X9500, DLA-X7500, DLA-RS620, and DLA-RS520 families. The source manual (v1.4) also documents commands for older DLA-HD/RS/X model variants; commands are tagged with applicable model families. Transport is binary hex on RS-232C (DB-9, 19200 8N1) and on TCP port 20554 with PJ_OK/PJREQ/PJACK handshake.

## Notes
- RS-232C uses a null-modem (cross-connected) DB-9 cable: pin 2 Rx, pin 3 Tx, pin 5 GND; pins 1, 4, 6-9 unconnected.
- LAN control requires TCP port 20554. After connect, projector replies `PJ_OK`; controller must reply `PJREQ` within 5s; projector replies `PJACK`; command must follow within 5s or connection closes. Connection closes 5s after response.
- Break ≥50ms in incoming serial data discards the in-flight command. Controller must wait for acknowledgement before next command.
- Power-off via the RC emulation command (`52 43 37 33 30 36`) must be sent twice with a short delay between to actually switch off the projector.
- Model applicability varies by command — the source tags each command with the supported DLA-HD/RS/X model families. The X9500/X7500/RS620/RS520 are the modern models; the older model families documented (HD350/550/750/950/990, X3/X7/X9/X30/X70/X90, RS10/15/20/25/35/40/45/50/55/60/65) share the protocol family.
- DHCP default Off; default IP 192.168.0.2; default subnet 255.255.255.0; default gateway 192.168.0.254.
- IR Code A = hex 73 (decimal 115); IR Code B = hex 63 (decimal 99).
